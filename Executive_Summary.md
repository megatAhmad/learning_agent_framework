## 🎯 Program Overview

The Microsoft Agent Framework Mastery Program is a structured, hands-on learning course designed to teach developers how to build production-ready multi-agent AI systems using Python. The program consists of **15 progressive learning steps**, each accompanied by an executable Jupyter notebook (.ipynb) file.

### Key Specifications

| Aspect | Details |
|--------|---------|
| **Format** | 15 Jupyter Notebooks (.ipynb files) |
| **Language** | Python only |
| **Development** | Local development (no cloud deployment) |
| **LLM Providers** | OpenRouter + Azure OpenAI |
| **Duration** | 4-6 weeks (full-time) or 12-16 weeks (part-time) |
| **Total Time** | ~1,000-1,200 minutes (~17-20 hours) |
| **Target Audience** | Python developers, data scientists, backend engineers |
| **Prerequisite** | Basic Python knowledge |

---

## 📚 The 15 Learning Steps

### Foundation (Steps 1-3) - ~2 hours
- **Step 1**: Setup & Environment Configuration
- **Step 2**: Your First Agent  
- **Step 3**: Multi-Turn Conversations

### Tools & Integration (Steps 4-7) - ~3.5 hours
- **Step 4**: Understanding Tools
- **Step 5**: Single Tool Integration
- **Step 6**: Multiple Tools
- **Step 7**: Custom Tools & API Integration

### Memory & Advanced (Steps 8-9) - ~2.5 hours
- **Step 8**: Memory & Sessions
- **Step 9**: RAG Implementation

### Workflows (Steps 10-11) - ~2.5 hours
- **Step 10**: Basic Workflows
- **Step 11**: Conditional Workflows

### Orchestration (Steps 12-13) - ~3 hours
- **Step 12**: Orchestration Patterns (Sequential, Concurrent, Handoff)
- **Step 13**: Advanced Orchestration (Group Chat, Magentic One)

### Production (Steps 14-15) - ~2.5 hours
- **Step 14**: Human-in-the-Loop
- **Step 15**: Production Features (Logging, Monitoring, Cost Tracking)

## 🛠️ Technical Stack

### Core Components
- **Framework**: Microsoft Agent Framework v1.0 (production-ready)
- **Language**: Python 3.9+
- **Interface**: Jupyter Notebooks
- **Async**: asyncio/async-await patterns

### LLM Providers
1. **OpenRouter** (Primary)
   - 100+ models available
   - Free tier with limits
   - Cost tracking built-in
   - Easy switching between models

2. **Azure OpenAI** (Secondary)
   - Enterprise-grade
   - Azure authentication
   - Production-ready infrastructure
   - Compliance certifications

### Supporting Technologies
- SQLite (for persistent memory)
- Pandas/NumPy (data processing)
- scikit-learn (similarity calculations)
- aiohttp (async HTTP requests)
- pydantic (data validation)
